# LLM-HPC Survey Resources

This repository provides supporting data, figures and references for the survey paper **“Exploring the Role of LLMs in HPC Programming: A Survey”**.

## Repository Structure

- **/data/**  
  Contains the raw data used in the survey, including:
  - `Research Papers.xlsx` – metadata on included papers
  - `Categorized.xlsx` – distribution of papers across categories.

- **/figures/**  
  Contains the figures used in the survey.

- **/papers/**  
  Contains bibliographic references (BibTeX format) for the 19 peer-reviewed papers included in the survey:
  - `references.bib`

## Usage

The materials in this repository are intended to support **reproducibility, transparency and extendibility** of the survey.  
Researchers can reuse the datasets, adapt the figures or extend the reference list with newly published work.
